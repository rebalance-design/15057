# Підключення форми до Google Таблиці

## 1. Створіть сховище

1. Створіть нову Google Таблицю для заяв.
2. Скопіюйте її ID з адреси між `/d/` і `/edit`.
3. Створіть окрему папку Google Drive для файлів `.p7s` та зображень.
4. Скопіюйте ID папки з її адреси.

## 2. Створіть Apps Script

1. Відкрийте `script.google.com` і створіть новий проєкт.
2. Вставте в нього вміст файла `google-apps-script.gs`.
3. У блоці `CONFIG` замініть:
   - `PASTE_GOOGLE_SHEET_ID_HERE`;
   - `PASTE_GOOGLE_DRIVE_FOLDER_ID_HERE`.
4. У налаштуваннях проєкту виставте часовий пояс `Europe/Kyiv`.

## 3. Розгорніть вебзастосунок

1. Натисніть **Deploy → New deployment**.
2. Тип: **Web app**.
3. **Execute as:** Me.
4. **Who has access:** Anyone.
5. Підтвердьте дозволи Google.
6. Скопіюйте URL, який закінчується на `/exec`.

## 4. Підключіть сайт

У файлі `script.js` замініть:

```js
const APPS_SCRIPT_URL = 'PASTE_GOOGLE_APPS_SCRIPT_WEB_APP_URL_HERE';
```

на отриманий URL вебзастосунку.

## 5. Перевірте

1. Надішліть тестову заяву без файла.
2. Переконайтеся, що рядок з’явився в таблиці.
3. Надішліть тест із `.p7s` або зображенням.
4. Перевірте файл у папці Drive і посилання в таблиці.

## Перед публічним запуском

Рекомендовано додати Cloudflare Turnstile. Поточна версія вже має honeypot, перевірку швидкості відправлення, обмеження повторних заяв і валідацію файлів, але Turnstile дасть сильніший захист від масового спаму.
